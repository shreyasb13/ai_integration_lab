# Google Gemini AI Chatbot

## Model Information
- **Model Name:** Gemini 3 Flash (Preview) 
- **Source:** Google Gemini API via Google AI Studio / Google Cloud  
- **Provider:** Google  

## Rationale for Model Selection
Gemini 3 Flash (Preview) was selected because it is designed for fast response times and efficient performance, which makes it well suited for chatbot applications. Since the goal of this project is to create a conversational chatbot using Python, the Flash model provides quick and coherent responses while maintaining good language understanding.

## Reflection on Responsible AI Use
Responsible AI use requires developers to consider transparency, bias, and appropriate use of generated content. When building this chatbot, it is important to recognize that generative AI models may often provide inaccurate responses. Users should be informed that the chatbot’s responses are AI-generated and should not always be treated as accurate information. Developers also have a responsibility to avoid exposing sensitive data when interacting with AI systems. Proper safeguards, such as limiting prompts that involve personal or harmful content and reviewing outputs when necessary, help ensure ethical usage. By being transparent about the system’s capabilities and limitations, developers can promote responsible interaction with AI-powered applications.
